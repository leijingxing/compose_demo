# AGENTS.md

## 项目说明
- 本项目为 Android Compose 学习 Demo。
- 代码需保持简洁、可读，便于初学者理解。

## 注释规范
- 每个类必须有中文注释。
- 每个变量都必须有中文注释：
  - 成员变量
  - 局部变量
  - Lambda 参数（必要时将 `it` 改为有意义的命名并添加注释）
  - 函数/构造函数参数
- Compose 相关函数建议使用 KDoc 简要说明用途与参数。

## 风格与编码
- 默认使用 ASCII；只有确有必要时才引入非 ASCII 字符。
- 注释要简短明确，避免冗长。
- 若不确定如何注释，优先使用“用途 + 作用范围”的描述。
